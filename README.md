Notes:
- Frontend is the page you open in the browser.
- Backend is the API endpoint used by the frontend.


How they work together
- The frontend is the user interface.
- The backend is the server that processes the data.
- When you use the app, the frontend sends a request to the backend.
- The backend returns the response, and the frontend displays it on the page.
